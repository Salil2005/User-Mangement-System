# User Management System with Admin Dashboard

## Project Overview

This project is a **Smart User Management System** built using Node.js.
It allows users to register, login, and view their dashboard, while an **admin panel** allows administrators to view and manage registered users.

The system demonstrates basic authentication flow, server-side routing, and user data handling.

## Features

* User Registration
* User Login Authentication
* User Dashboard with profile information
* Admin Login
* Admin Dashboard to view registered users
* Display user details in card format

## 🛠 Tech Stack

* Node.js
* PostgreSQL
* HTML
* CSS
* JavaScript
